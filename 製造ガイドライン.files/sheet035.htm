機能 Oracleネットワーク設定                                                        
■Oracleネットワーク設定
tnsnames.ora(C:\oracle\ora92\network\ADMIN\tnsnames.ora)を以下のように書き換える。
※他人のtnsnames.oraで上書きしてもよい。
# TNSNAMES.ORA Network Configuration File: C:\oracle\ora92\network\admin\tnsnames.ora
# Generated by Oracle configuration tools.              
                         
INSPIRE2 =                      
  (DESCRIPTION =                    
    (ADDRESS_LIST =                  
      (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.1.5)(PORT = 1521))    
    )                        
    (CONNECT_DATA =                  
      (SID = orcl)                    
      (SERVER = DEDICATED)                
    )                        
  )                        
TNS接続文字列       INSPIRE2？
Oracleデータベースサーバのホスト名 192.168.1.5
Oracleリスナーのポート番号     1521  
OracleインスタンスのSID     orcl  
■説明
Oracleは複数の「インスタンス」で構成されている。
「インスタンス」は複数のメモリ領域と複数のプロセスで構成される。
リスナーとはいわゆる「デーモン(daemon)」のことであり、接続するインスタンスをSIDによって切り替えているらしい。
ちなみにGUIで設定することも可能であり、
その場合は「Net Configuration Assistant」を選択する。
このツールで設定した内容は単に「C:\oracle\ora92\network\ADMIN\tnsnames.ora」に保存さるだけである。
