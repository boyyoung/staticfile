PG プロパティデータバンク様 @Property ASPサービス 製造資料      
   
機能 メニューオプション関連                                                        
   
 
 
  各画面ではラジオボタンやチェックボックス、登録・修正・削除・印刷ボタンなどがある。  
  それらは標準機能として使用されたりカスタマイズによって追加されてオプションをつけることで表示・非表示することがある。  
  本機能は契約IDによって使用するパッケージや機能が異なり、オプション機能もそれぞれ異なる。  
   
  ※JSP上でOPTIONを判定したい時  
  　たとえば、ある画面には『原因』というラジオボタンがあり、それには『bm_claim_cause』というオプションを設定したとき、アクセスした契約IDのユーザーがそのオプションをもっているかどうかをチェックし、  
  　その機能を表示したり、日表示したりするためには、下記のような記述を行う。  
   
  　●OｌｄのJSPソース上でチェックする方法  
          <% if (((com.property.data.common.PrDtLoginInfo)request.getSession().getAttribute("s_logininf")).getOption().containsKey("bm_claim_cause")) { %>  
          <input type="radio" name="v_syukei_kbn" value="4">  
          <span class="px-12"><%= i18n("原因") %></span>  
          <% } %>  
   
   
   
  　●NEWのJSPソース上でチェックする方法  
          <logic:isOption property="bm_claim_cause">  
          <input type="radio" name="v_syukei_kbn" value="4">  
          <span class="px-12"><%= i18n("原因") %></span>  
          </logic:isOption>  
   
   
  ※　JAVA上でOPTIONを判定したい時  
   
  　●NEWのJAVA上でチェックする方法  
      if (super.isOption(PrConstant.bm_claim_cause)) {  
   
      }  
  　●OLDはそれぞれのソースを参考にしてください。  
   
  ※　OPTIONの設定は　ASPDBA.ASP_OPTIONSテーブルで管理及び登録する
